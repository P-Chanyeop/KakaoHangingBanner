fix: 패키지 구조 및 API 엔드포인트 문제 해결

## 변경 사항
- 패키지 구조 문제 해결: 메인 애플리케이션 클래스와 컨트롤러 패키지 간 불일치 해결
- 컴포넌트 스캔 범위 확장: `@ComponentScan`, `@EntityScan`, `@EnableJpaRepositories` 어노테이션 추가
- 중복 컨트롤러 비활성화: URL 매핑 충돌 해결을 위해 중복 컨트롤러 비활성화
- CORS 설정 개선: 보안 문제 해결을 위한 CORS 설정 수정
- H2 데이터베이스 방언 수정: MySQL 방언에서 H2 방언으로 변경
- API 테스트 페이지 추가: API 엔드포인트 테스트를 위한 전용 페이지 구현
- 게시대 등록 UI 개선: 지도 아래에 인라인 폼 배치로 사용성 향상
- README.md 업데이트: 프로젝트 구조, 설치 방법, 주요 기능 등 최신 정보 반영

## 이유
- 서버 시작 시 발생하는 빈 의존성 주입 오류 해결
- URL 매핑 충돌로 인한 애플리케이션 시작 실패 문제 해결
- CORS 설정 오류로 인한 API 접근 문제 해결
- 사용자 경험 개선을 위한 UI 변경

## 테스트
- 서버 정상 시작 확인
- API 엔드포인트 정상 작동 확인
- 게시대 등록 및 조회 기능 정상 작동 확인
- CORS 설정 정상 작동 확인
